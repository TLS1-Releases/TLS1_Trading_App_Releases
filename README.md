## 🌟 Tính năng nổi bật

- 🤖 **Bot Giao Dịch Tự Động:** Tích hợp các chiến thuật độc quyền như **Bot EMA200** và **Bot SMC - OB**.
- 🔐 **Bảo mật tuyệt đối:** Hoạt động dưới dạng Portable App (Không cần cài đặt). Mọi thông tin API Key đều được mã hóa và lưu trữ cục bộ 100% trên máy tính của bạn (tại vùng an toàn AppData).
- 🎛️ **Quản lý Đa Tài Khoản (Sub-accounts):** Hỗ trợ thêm/xóa và luân chuyển mượt mà giữa hàng loạt tài khoản phụ (Sub-account) chỉ với 1 cú click chuột.
- 📊 **Kiểm soát thông minh:** Hệ thống Audit, Reset vốn gốc và biểu đồ theo dõi nến thời gian thực cực kỳ trực quan.

## 📥 Hướng dẫn Tải và Sử dụng

Vì lý do bảo mật và tính tiện dụng, phần mềm được thiết kế dưới dạng **Portable (Chạy trực tiếp không cần cài đặt)**.

1. Truy cập vào mục **[Releases](https://github.com/traderlaso1wolfcapital-creator/TLS1_Trading_App_Releases/releases)** ở thanh menu bên phải màn hình.
2. Tìm đến phiên bản **Latest (Mới nhất)**.
3. Tải xuống file chạy tương ứng với hệ điều hành của bạn:
   - 🪟 Cho Windows: Tải file `TLS1_Trading_Setup.exe`
   - 🍎 Cho MacOS: Tải file `TLS1_Trading_Mac_Universal.dmg`
4. Copy file vừa tải ra màn hình Desktop, nháy đúp chuột để mở và bắt đầu trải nghiệm!

> [!WARNING] 
> **Lưu ý an toàn:** Tuyệt đối không cung cấp API Key (có quyền Rút tiền) cho bất kỳ phần mềm nào. Chỉ cấp quyền **Read (Đọc)** và **Trade (Giao dịch)** cho API Key sử dụng trong App này.
